# PageForge

Plataforma de criação automatizada de sites com IA.

## Estrutura do projeto

| Pasta | Responsável | Descrição |
|-------|------------|-----------|
| `app/` | Dev 1 | Landing page, painel do cliente, painel admin |
| `app/briefing/` | Dev 2 | Formulário de briefing |
| `app/preview/` | Dev 3 | Preview do site gerado pela IA |
| `components/layout/` | Dev 1 | Header, sidebar, footer |
| `components/briefing/` | Dev 2 | Componentes do formulário |
| `components/ai/` | Dev 3 | Componentes de IA e preview |
| `public/references/` | Dev 2 | Galeria de referências visuais |

## Como rodar

npm install
npm run dev

Acesse: http://localhost:3000
